Package fmt

import "fmt"

Types

trait Writer {
	write([]char)
}

trait Show {
	show(Writer)
}

Functions

fn write(w Writer, shows ...Show)

Write the arguments into the writer 

fn writeln(w Writer, shows ...Show)

Write the arguments into the writer with a following newline 

fn print(shows ...Show)

Print the arguments to standard output 

fn println(shows ...Show)

Print the arguments to standard output with a following newline 

fn eprint(shows ...Show)

Print the arguments to standard error 

fn eprintln(shows ...Show)

Print the arguments to standard error with a following newline
