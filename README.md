## Quick Start
```console
$ cc -o first first.c # On Linux/macOS
$ cl first.c          # On Windows
$ ./first
```

## Tests
```console
$ ./first -t # Interactive mode
$ ./first -T # Non-interactive mode
```

## Platforms supported
- x86_64 Linux
- x86_64 Windows
- ARM64 macOS
