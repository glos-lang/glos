## Quick Start
Install `clang`

```console
$ clang -o first first.c     # On Linux/macOS
$ clang -o first.exe first.c # On Windows
$ ./first
```

## Tests
```console
$ ./first -t # Interactive mode
$ ./first -T # Non-interactive mode
```

## Platforms supported
- x86_64 Linux
- x86_64 Windows
- ARM64 macOS
